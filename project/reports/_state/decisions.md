# 📝 Karar Kayıtları (Decisions)

> Audit sırasında alınan kararlar buraya kaydedilir.
> Özellikle "Güvenli Fix Sınırları"nı aşan değişiklikler, onaylanan/red edilen riskli fix'ler ve kullanıcıdan gelen talimatlar.

---

## Format

```markdown
## Karar #KARAR-[SIRA]
- **Tarih:** YYYY-MM-DD HH:mm
- **Konu:** [kısa başlık]
- **Karar Veren:** [agent adı / kullanıcı]
- **Gerekçe:** [neden bu karar alındı]
- **Sonuç:** [ne yapıldı / yapılmadı]
- **Etkilenen Dosyalar:** [path'ler]
```
